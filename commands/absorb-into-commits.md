# absorb-into-commits: Absorb changes into earlier commits

Absorb uncommitted changes or recent fix commits into the logically
correct earlier commits via interactive rebase.

## Usage

```
/absorb-into-commits
```

## Execution

1. Read `skills/absorb-into-commits/SKILL.md` from this plugin's directory.
2. Follow the skill instructions to absorb changes into earlier commits.
